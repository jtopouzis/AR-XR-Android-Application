# AR-XR-Android-Application

A Demo Augmented Reality (AR/XR) application developed using Unity, focused on delivering immersive and interactive experiences in real-world environments.

---

## 🚀 Tech Stack
- Unity Engine
- AR Foundation
- ARCore (Android)
- Meta XR SDK
- Universal Render Pipeline (URP)
- TextMeshPro
- C#

---

## 🧩 Features
- Plane detection & real-world surface tracking  
- Object placement in AR space  
- Real-time interaction with 3D objects  
- Light estimation for realistic rendering  
- Mobile-optimized performance (URP)  
- Dynamic in-world text rendering
